### Маркированный список

Маркированный список создается с помощью символа * или -.

* Элемент списка 1
* Элемент списка 2
* Элемент списка 3

### Нумерованный список

Нумерованный список создается с помощью цифры и точки.

1. Элемент списка 1
2. Элемент списка 2
3. Элемент списка 3
